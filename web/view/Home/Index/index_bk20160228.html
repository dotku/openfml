关于我们

范米粒 (fnmili.com) 是一个美国代购服务小组，该小组是由两个软件工程师发起的。美国代购员 Jay 是硅谷一家公司的全栈开发工程师，中国代理 贾德荃 是福州的前端开发工程师。

愿为您服务:)

关于服务

我们提供美国直邮服务，不定期（一般是周末）采集美国的现场产品，您可以加入到我们的 QQ 群，参与我们的群互动，可以个人，或者小组预约直播现场代购，或者现场外的代购求购。

关于收费

  • 代购: 我们仅收产品 7% 的服务费（不涉及税收、运费、关税、汇率等）。
  • 代刷: 由于不涉及包裹查收等步骤，代刷可以更低一些，我们的代刷服务收取 5% 的服务费（不涉及税收、运费、关税、汇率等）。

## 费用与支付

### 费用

  • (物品价格 + 实时汇率) x 107% + 运费 (有包税和不包税渠道)

### 运费

我们并非物流公司，我们只是希望服务好我们的客户，所以不希望花更多的时间去处理物流，专业术语一点叫做"物流优化"。我们精挑细选了几家合适的华人快递公司作为我们的物流服务商，他们都比美国本地邮局要便宜 75 折以上，包括清关和不清关渠道。

清关包税渠道
  • 食品、保健品、婴儿用品、衣服、玩具、安全座椅等 4.5 / 磅
  • 护肤品、化妆品、小电器 6.5 / 磅
  • Coach、MK、KS 等品牌包包 7 / 磅
  • 不在上述列表中 7 / 磅
  • 提供全美境内包括全球 218 个国家地区到门服务，价格是美国邮政的 75 折
额外关税
  • 装品、眼镜、手表等 $100 一下补充关税 $2，$200 以下的关税 $6， $200 以上的按 3% 算
  • 香水一瓶另交关税 $9
不清关不包税渠道
3 USD / 磅 (6 磅起计算)
无税清关方式
(*) 不清关渠道，有一定机率被关税，如果被关税，需要补交关税；由物流公司负责补税，无需客户前往。
